This program is the product of a project done by Master forensic students in the UvA.

forensic power for good. In essence, we have collected all collectable artifacts from discord
and present it here in a user friendly overview. On the right hand side you can find links about us and
about the project. To restart the project use:

export FLASK_APP=serv.py ; flask run

where serv.py is the hard link to the serv.py file.
